# Company Directory Flutter App

A Flutter-based mobile and desktop app for managing company information.  
This project includes user authentication, company listings, detailed views, and feedback functionality.

## Features

- **User Authentication**: Login and signup screens with secure authentication.  
- **Company Listings**: Browse and search companies with detailed information.  
- **Company Details**: View individual company details, including description and contact info.  
- **Feedback System**: Users can submit feedback on companies.  
- **Cross-Platform**: Runs on Android, iOS, Windows, Linux, and macOS.  
- **API Integration**: Connects to backend services for dynamic data.  

## Tech Stack

- Flutter & Dart  
- RESTful API for backend integration  
- SQLite or remote database for storage
